# MiRoot V2.1 正式版发布说明

## 下载

- APK：[MiRoot-Releases v2.1.0](https://github.com/wmqc97/MiRoot-Releases/releases/tag/v2.1.0)

## 更新摘要

### 新增

- **WebDAV 配置云备份**（状态页入口）：坚果云等 WebDAV 手动上传/下载；本机 ZIP 导出/导入；`manifest.json` + `backup_label.txt` 记录导出时间与包大小；恢复前弹窗展示备份信息，降低误覆盖风险
- **车控**：桌面小组件；车辆历史 SQLite 与图表页；背屏地图/油价；可自定义车模与底部按钮
- **歌词**：`LyricsApiEndpoints` / `LyricsPlatformClients` 等多源 API 重构

### 优化

- 主 Tab / 二级页顶栏与状态栏颜色统一；标题字号一致
- 音乐页标题改为「音乐」

### 修复

- 背屏录屏/截图导出、Shizuku 相关稳定性与文档
