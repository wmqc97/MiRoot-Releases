# MiRoot-Releases

本仓库仅用于发布 **MiRoot 正式版安装包**，不包含源码。

## 简介

[MiRoot](https://github.com/wmqc97/MiRoot) 是面向小米 17 Pro 系列等背屏手机的背屏增强工具：背屏投屏、歌词投影、星瑞车控、充电动画自定义、WebDAV 配置云备份等。

## 最新版本

| 版本 | 说明 |
|------|------|
| **V2.1** | WebDAV 云备份；车控小组件与车辆历史；界面统一；歌词 API 重构 |

请到 [Releases](https://github.com/wmqc97/MiRoot-Releases/releases) 下载最新 `MiRoot_*_Release.apk`。

## 安装

1. 下载对应版本的 Release APK
2. 安装后打开 MiRoot，在「状态」页完成 Root / Shizuku 授权
3. 应用内「状态 → 配置云备份」可配置 WebDAV（可选）
